# **App Name**: TabulaNote

## Core Features:

- Full-Screen Editor: Full-screen text editor in a new tab for distraction-free note-taking.
- Auto-Save: Automatic saving of notes to local storage.
- Text Formatting: Plain text formatting options (bold, italic, underline).
- Export Notes: Option to export notes as text files.
- AI Summarizer: AI tool for generating summaries of long notes, allowing for quick review.
- Keyboard Shortcuts: Keyboard shortcuts for common actions (e.g., save, new note).

## Style Guidelines:

- Primary color: Soft blue (#78A0DC) to evoke calmness and focus, avoiding overly stimulating hues. This color takes inspiration from the clear, expansive nature of a blank digital canvas, and also aligns to familiar, analog precedents in notetaking like legal pads.
- Background color: Light gray (#F0F2F5), nearly white, for a clean and readable surface in a light scheme.
- Accent color: Muted violet (#946CB2) to add a touch of uniqueness while remaining subtle.
- Body and headline font: 'Inter', sans-serif, for clear and legible text.
- Minimalist design with a focus on the text editor area.
- Subtle fade-in/out transitions for a smooth user experience.
